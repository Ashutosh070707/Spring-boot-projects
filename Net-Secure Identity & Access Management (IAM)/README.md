## Steps to create this project:

### Create a maven project by adding depndencies (Spring web, Spring Data Neo4j, Spring Boot DevTools, spring-boot-starter-validation) and configure your neo4j database (uri, username, password) in application.properties file.

### Create entities - User, Group, Resource

### Create Reopisitories - UserRepository, Group Repository, ResourseRepository

### Add all the contraints required for each entity properties in the database before adding any data. You can add indexes later also.

### Add GlobalExceptionhandler to handle any exception thrown while application is running.
